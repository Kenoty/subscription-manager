# SubscriptionManager

Subscription Manager — платформа для управления подписками на стриминговые и онлайн-сервисы. Позволяет пользователям отслеживать все свои подписки в одном месте, контролировать расходы, получать уведомления об истечении срока подписки и предстоящих платежах, а также управлять устройствами на которых используются подписки.
